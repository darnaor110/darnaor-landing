## What You're Building

A Google Sheet with three sections per month:

**SECTION 1: Content Planning Table**
- One row per video
- Columns: Video #, Post Date, Status, Funnel Type, Script Angle, Content Type, Hook Stack, Video Format, CTA, Editing Style Notes
- Then: Script lines + Filming Notes + Editing Notes (one set of 3 columns per line)

**SECTION 2: Monthly Calendar**
- Traditional calendar grid (Mon–Sun)
- Shows when each video posts
- Visual layout reference

**SECTION 3: TOFU/MOFU/BOFU Ratio Check**
- Auto-counts how many of each type you have
- Compares to target
- Flags if you need to adjust

---

## The Workflow (Once Built)

1. **Planning:** Fill in Hook Stack, Script Angle, Format, CTA (takes ~5 min per video)
2. **Scripting:** Write script lines (1 sentence each) + filming notes + editing notes (takes ~10–15 min per video)
3. **Filming:** Shoot according to filming notes; update Status → "Filmed"
4. **Editing:** Edit according to editing notes; update Status → "Edited"
5. **Uploading:** Post video; update Status → "Posted"
6. **Monthly review:** Check ratio; adjust next month's content type mix if needed

## Ava's 7 Factors (All Captured in the Sheet)

✅ **Topic** — Your idea/value  
✅ **Hooks** — Verbal + Written + Visual  
✅ **Value** — The meat of the video (script lines)  
✅ **Script Angle** — One of 7 types (column E)  
✅ **CTA** — Call to action (column K)  
✅ **Format** — How it's filmed (column J)  
✅ **Editing Style** — Colors, fonts, B-roll, audio (column L)  

---

## Script Structure (One Row Per Take)

Each script line = ONE SENTENCE = ONE TAKE

Example:
```
Script: "Scrambled eggs cook fast."
Filming Notes: "Close-up of eggs in pan, scrambling with fork, white background"
Editing Notes: "Text '❌ QUICK' overlay, fade in 0.3s, stay 1.5s"
```

Because you struggle with long lines, this keeps everything filmable in single takes.
