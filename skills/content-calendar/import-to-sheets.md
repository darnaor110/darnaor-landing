# How to Import June 2026 Skeleton into Google Sheets

**What you have:** `june-2026-import.csv` — a pre-built CSV with 8 videos scaffolded for June

**What you're doing:** Import this into your Google Sheet to instantly create the skeleton

**Time needed:** 3 minutes

---

## Step 1: Download the CSV File

- File location: `c:\Users\darna\claudeproject\skills\content-calendar\june-2026-import.csv`
- Download it to your computer (or just have it ready)

---

## Step 2: Create New Tab in Google Sheets

1. Go to your Google Sheet: https://docs.google.com/spreadsheets/d/1HP0kJtm_kgHA64N55FqP1cHcjXRsDgbZt77YHU4y4dE/edit?usp=sharing
2. At the bottom, click the **"+"** button (next to existing tabs)
3. A blank tab appears
4. Right-click the new tab → **Rename** → Type `June 2026` → Press Enter

**Result:** You now have a blank "June 2026" tab open

---

## Step 3: Import the CSV

1. Make sure you're on the **June 2026** tab
2. Go to **File** → **Import**
3. Choose **Upload** tab
4. Click **Select a file from your computer**
5. Navigate to: `c:\Users\darna\claudeproject\skills\content-calendar\june-2026-import.csv`
6. Select it → **Open**

---

## Step 4: Choose Import Settings

A dialog appears asking "Import action":

- **Replace spreadsheet** — No, leave unchecked
- **Replace current sheet** — Check this (replaces the blank June 2026 tab with the CSV data)
- **Append to current sheet** — No
- **Create new sheet** — No (already have June 2026)

Click **Import data**

---

## Step 6: Set Up Formatting (Optional But Recommended)

The CSV imports data only — formatting needs manual setup. Do these to match design:

### Freeze Row 1
1. Click Row 1
2. **View** → **Freeze** → **1 row**

### Set Column Widths
1. Select all columns (click top-left corner)
2. Right-click → **Column width**
3. Set:
   - A–C: 70px
   - D–F: 100px
   - G–I: 120px
   - J–K: 100–120px
   - L: 150px
   - M onwards: 180px

### Enable Text Wrapping
1. Select columns M onwards (script/notes columns)
2. **Format** → **Wrap text**

### Add Dropdowns (Data Validation)

These are important — they keep data clean.

**Column C (Status):**
1. Click column header **C**
2. **Data** → **Data validation**
3. List: `Ready for filming,Filmed,Edited,Posted`
4. Check "Show dropdown arrow" → **Done**

**Column D (Funnel Type):**
1. Click column **D**
2. **Data** → **Data validation**
3. List: `TOFU,MOFU,BOFU`
4. Check dropdown → **Done**

**Column E (Script Angle):**
1. Click column **E**
2. **Data** → **Data validation**
3. List: `Tutorial,Comparison,Mythbust,Do's vs Don'ts,Tip/Hack,Transformation,Challenge`
4. Check dropdown → **Done**

**Column F (Content Type):**
1. Click column **F**
2. **Data** → **Data validation**
3. List: `Educational,Storytelling,Authority`
4. Check dropdown → **Done**

**Column J (Video Format):**
1. Click column **J**
2. **Data** → **Data validation**
3. List: `Talking Back and Forth,Visual,Voiceover,Multitasking,Setting Change,Shot/Angle Change,Clone,Whiteboard,Q&A,Green Screen,Reaction`
4. Check dropdown → **Done**

**Column K (CTA Type):**
1. Click column **K**
2. **Data** → **Data validation**
3. List: `Follow,Freebie,Paid Offer,Engagement,None`
4. Check dropdown → **Done**

## Troubleshooting

**"I don't see the data after import"**
→ Make sure you checked "Replace current sheet" before importing

**"The dates look weird"**
→ Google Sheets might auto-format dates. You can change them to your preferred format. Just click a date cell → **Format** → **Date** → Choose format

**"Dropdowns aren't working"**
→ Make sure you followed the Data Validation steps exactly. The list items must match exactly (including punctuation).

**"Some columns are too narrow"**
→ Right-click column → **Column width** → increase the number
