# Ava Yuergens — Master Action Points List

> Every concrete, actionable step from all 13 videos. Organized by category.

---

## FOUNDATION — SET UP ONCE

- [ ] Write your main broad niche (fitness, business, nutrition, etc.)
- [ ] Write 5–7 subniches under your main niche
- [ ] Write your occupation/title as a keyword
- [ ] Create a list of niche keywords combining: niche + subniches + occupation title
- [ ] Choose 2–3 brand colors
- [ ] Choose 3 fonts: primary (hooks/bold text), secondary (captions), optional accent (annotations)
- [ ] Choose 2–3 consistent visual types for your videos (B-roll, screenshots, screen recordings, charts, etc.)
- [ ] Set these as your permanent editing style — apply to every video

---

## PROFILE OPTIMIZATION — SET UP ONCE

- [ ] Set username to your full name or business name only
- [ ] Write tagline as: Name | [keyword people search to find someone like you]
- [ ] Take a close-up profile picture with a solid background — use PFP Maker to match your brand colors
- [ ] Write a 4-line bio that tells who you are and what you do in 5 seconds
- [ ] Set one single link in bio (landing page or freebie) — remove Linktree if you have it
- [ ] Set up ManyChat for all lead CTAs (never use "link in bio" for offers)

---

## RESEARCH — DO THIS MONTHLY

- [ ] Create a separate Instagram "dummy" account for research only
- [ ] Reset its algorithm: Settings → Content Preferences → Reset Suggested Content
- [ ] Run keyword research for every keyword on your list:
  - Open Instagram Explore → search each keyword → click Reels tab → save videos following the 5x rule
  - Search multiple variations of each keyword
- [ ] Run creator research for every keyword:
  - Search keyword → click Accounts tab → find creators you like → screenshot their accounts
  - For each creator: use Sortfeed Chrome extension → sort by views → save outliers following the 5x rule
  - Check "Suggested Accounts" from the best accounts you find
  - Check the "Following" list of the best creators to find other niche accounts
- [ ] After keyword + creator research, scroll the Reels recommendation tab on the dummy account — save hidden gem outliers you couldn't find in regular research
- [ ] End goal: have a bank of saved videos with proven viral verbal/written/visual hooks following the 5x rule

---

## IDEATION — DO THIS MONTHLY (per batch)

- [ ] Decide your posting amount for the month (Easy / Moderate / Hard)
- [ ] Apply the right content split rule:
  - New account: 90% proven outliers, 10% original experiments
  - Established account: 70% proven outliers, 20% double-downs, 10% original experiments
- [ ] For each video idea, create a **Hook Stack** (verbal + written + visual hook — each from different outlier videos, not the same source)
- [ ] For each video idea, determine the value: what knowledge or story will you use?
- [ ] For each video idea, choose a script angle (Tutorial / Comparison / Mythbust / Do vs Don't / Tip/Hack / Transformation / Challenge)
- [ ] For each video idea, decide: educational approach or storytelling approach?
- [ ] For each video idea, choose a CTA (Follow / ManyChat freebie / ManyChat paid offer / Engagement / None)
- [ ] For each video idea, choose a filming format (avoid sitting and talking)
- [ ] For each video idea, decide the TOFU/MOFU/BOFU type and match the CTA accordingly
- [ ] Plan your monthly split: the right number of TOFU, MOFU, and BOFU videos based on your follower level

---

## SCRIPTING

- [ ] Build a Google Doc bank of script structures — reuse these constantly
- [ ] For each video: find a viral outlier script structure → transcribe it via gettranscribe.ai → templatize with ChatGPT ("make this a fill-in-the-blank template for any niche") → save to your Google Doc bank
- [ ] When scripting: use your hook stack + your chosen script structure + your own expertise/stories for the value
- [ ] Always use **measurable value** — hyper-specific, actionable, clear. Never fluffy, vague, or generic.
- [ ] Format your script: black = script text | red = shot/filming instructions | green = editing notes for editor
- [ ] For storytelling videos: plan every shot needed + editing notes (past images, B-roll, emotional background music)
- [ ] For complex videos: add visuals/B-roll references under each bullet in green

---

## FILMING

- [ ] Equipment: phone + Amazon tripod + Amazon microphone (no fancy gear needed)
- [ ] Film one bullet at a time
- [ ] Choose an engaging format — never "sit and talk" unless combined with other techniques
- [ ] Create one big folder per content batch; one sub-folder per individual video
- [ ] Upload all shots for each video into its individual folder immediately after filming

---

## EDITING

- [ ] Edit yourself using CapCut or hire an editor (if you're a business owner, hiring is recommended)
- [ ] Apply your established editing style: brand colors, primary/secondary fonts, consistent visual types
- [ ] For storytelling videos: add an emotional background song — research other storytelling videos for inspiration on what hits

---

## UPLOADING

- [ ] Follow the caption structure:
  - Line 1: Written hook of the video
  - Double space
  - Line 2: CTA of the video
  - Double space
  - Line 3: 3–5 hashtags (used as keywords for discovery, not for performance)
- [ ] Upload according to your set posting schedule
- [ ] For carousels: add a trending audio to give it a chance of appearing in the Reels tab

## CAROUSELS (specific actions)

- [ ] Try all 8 carousel types: Tutorial, Do vs Don't, Educational Tips, Storytelling, Transformation, Problem/Solution, Lists, Types
- [ ] Fast track to a viral carousel: find your most viewed Reel → turn it into a carousel in Canva → use same verbal hook as cover slide → post
- [ ] Always add trending audio to every carousel
- [ ] As a new creator: focus on Reels first. Add carousels as your audience grows.
- [ ] As an established creator: use carousels to connect with and convert your existing audience

## ANALYSIS & OPTIMIZATION (per month / per batch)

- [ ] After reaching 1K, study your top 5 best-performing and 5 worst-performing videos using the Edits app
- [ ] Identify your outliers (videos with 5x your average views)
- [ ] Break down each outlier's 7 components: topic, hooks, value, script angle, CTA, format, editing style
- [ ] Double down on each outlier using one of the 3 methods:
  - Method A: Keep most components, adjust value slightly
  - Method B: Keep format/structure, change other components
  - Method C: Keep all components, change only the filming format (most effective)
- [ ] Templatize your own viral outliers and add them to your 20% content slots

## SERIES CONTENT

- [ ] Pick a niche/topic for a series
- [ ] Choose a series type: Daily Series / Progress Update Series / Lesson Series
- [ ] Plan number of episodes + outline each episode's premise
- [ ] Set a posting schedule to stay accountable
- [ ] Leave cliffhangers at the end of each episode to drive follows
- [ ] Use engaging filming formats: shot/angle change, visual, voiceover, or setting change

## AI WORKFLOW SETUP

- [ ] Choose 1–3 LLMs to use (Ava uses Claude + Acquisition AI)
- [ ] Introduce yourself to your LLM: your role, business, goals, and what you want from it
- [ ] Ask the AI if it has clarifying questions
- [ ] Upload all your past high-quality work as data sources
- [ ] Create a master prompt using Dan Martell's method: "I want to create a master prompt for my role as [role]. Ask me all the questions you need to know." → answer questions → refine → save as PDF
- [ ] Start every new topic in a new chat (Hormozi rule)
- [ ] Use the prompt formula every time: Role + Context + Command + Format
- [ ] Install Whisper Flow for voice typing (saves ~1 hour/day of typing)
- [ ] Identify all roles in your content workflow (ideator, editor, copywriter, etc.)
- [ ] Break each role into individual workflows → break each workflow into individual actions
- [ ] Build a Skill markdown file for each workflow using this prompt: "I'm going to give you an SOP for a specific workflow in my business. I want you to turn it into a Skill markdown file. The Skill file should include: a clear title and one-sentence description; the exact trigger; every single step in observable language; decision points; what a perfect output looks like; common mistakes to avoid; what to do if something goes wrong. Here is my SOP: [paste SOP]"
- [ ] Test and refine each Skill file until the output equals or exceeds your own quality
- [ ] Connect agents in a chain (Agent 1 output → Agent 2 input) to automate entire workflows

## STANDING OUT IN THE AI ERA

- [ ] Build your own original series that no one can copy
- [ ] Develop your own viral filming format to double down on consistently
- [ ] Add social proof to scripts (personal results or client results)
- [ ] Add personal stories and experiences to every script — things AI can't generate
- [ ] Add your personality — don't sound like AI or like everyone else
- [ ] Be first to post new viral formats in your niche
- [ ] Never copy a video word for word — use structures and hooks, fill with your own content

## MINDSET RULES (non-negotiables)

- [ ] Never post content you will regret — even if it would go viral
- [ ] Only post about topics where you are genuinely an expert
- [ ] Use your own knowledge and experience as the primary value source
- [ ] Credit external informational sources if you use them
- [ ] Prioritize educational, valuable, vulnerable, personal, raw, real content — these get the best quality followers
- [ ] Never outsource content or business decision-making to AI
- [ ] Never give up — the only way to fail is to stop posting
- [ ] Be patient — growth timelines vary wildly; one video can change everything
