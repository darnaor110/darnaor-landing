**Core pillars to apply when creating content:**

1. **Three content types (always mix all three):**
   - Educational (BOFU) — measurable, specific value → builds authority → use for leads
   - Storytelling/Personal (MOFU) — raw, real, vulnerable → builds connection → follow CTA only
   - Authority (TOFU/trust) — results and proof → builds trust → use for lead/offer CTAs

2. **Hook Stack (every video):** Verbal hook + Written hook + Visual hook — all from proven viral outliers following the 5x rule (5x views vs. creator's following count)

3. **7 video factors:** Topic → Hooks → Value → Script Angle → CTA → Format → Editing Style

4. **Script structures:** Find viral outlier → transcribe (gettranscribe.ai) → templatize (ChatGPT) → fill with own content. Prefer complex/high-effort structures for better follower conversion.

5. **70/20/10 rule (established accounts):** 70% proven outlier hooks/topics, 20% doubling down on own viral content, 10% original experiments

6. **Doubling down:** Recreate viral videos with small tweaks. Most effective: keep all 7 components, change only the filming format. This produced 12M views + 83K followers in 90 days.

7. **Never fluffy value.** Always measurable, specific, actionable. Content must make someone's life better.

8. **Series content** = fastest growth strategy. Leave cliffhangers. Apply Netflix drip model.

9. **ManyChat** instead of "link in bio" for all lead CTAs.

10. **Caption structure:** Line 1 = written hook | Line 2 = CTA | Line 3 = 3–5 hashtags (as keywords)

**Key tools:** Sortfeed (find viral outliers), Edits app (analytics), gettranscribe.ai (transcription), ManyChat (lead CTAs), CapCut (editing)
