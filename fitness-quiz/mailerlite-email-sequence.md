# MailerLite Email Sequence — Dad Naor Fitness Funnel
# 5 emails over 5 weeks

---

## HOW TO SET UP IN MAILERLITE

1. Go to **Automations** → **Create new automation**
2. Trigger: **"When subscriber joins a group"** → select your quiz group (`152578390496380004`)
3. Add emails at these delays (from signup):
   - Email 1: **Immediately** (delay = 0 — but this is sent by n8n, not MailerLite. Just use MailerLite for emails 2–5)
   - Email 2: **Delay 7 days**
   - Email 3: **Delay 14 days**
   - Email 4: **Delay 21 days**
   - Email 5: **Delay 28 days**

> Note: Email 1 (the plan itself) is sent by your n8n workflow — it's richer HTML that MailerLite's builder can't generate dynamically per-user. Emails 2–5 can be built directly in MailerLite.

## EMAIL 2 — Week 1
**Subject:** למה רוב הדיאטות נכשלות (ולמה שלך לא תיכשל)
**Delay:** 7 days after joining

---

שלום {$name},

שבוע עבר מאז שקיבלת את התוכנית שלך.

אני רוצה לשאול אותך משהו ישיר:

**האם אכלת לפי התפריט?**

אם כן — מעולה. ממש.
אם לא — זה בסדר גמור. וזה בדיוק בגלל זה אני כותב לך היום.

---

**הסיבה האמיתית שדיאטות נכשלות לא קשורה לרצון.**

היא קשורה לכך שאנשים מנסים לשנות הכל בבת אחת, בלי להבין *למה* הם עושים מה שהם עושים.

הם יודעים *מה* לאכול. הם לא מבינים *למה* זה עובד.

אז בשבועות הקרובים אני אשלח לך סדרה של פרקים קצרים מהסדנה שלי — כאלה שמסבירים את הביולוגיה מאחורי התוכנית שלך.

**השבוע: פרק 1 — מה זה TDEE ולמה זה המספר הכי חשוב בחיים שלך**

[▶️ לצפייה בפרק 1 — לחץ כאן](VIDEO_LINK_WEEK1)

פרק קצר. 10–12 דקות. מסביר בדיוק איך חישבתי את התוכנית שלך.

## MAILERLITE CUSTOM FIELDS TO CREATE

Go to **Subscribers → Custom fields** and add these (they're set by the quiz automatically):

| Field name | Type |
|---|---|
| `name` | Text |
| `tdee` | Number |
| `target_calories` | Number |
| `protein_target` | Number |
| `target_weight` | Number |
| `kg_to_lose` | Number |
| `pace` | Text |
| `snack_spare` | Number |
| `training_days` | Number |
| `training_level` | Number |

In email copy, use MailerLite's merge tags like `{$name}`, `{$kg_to_lose}` etc.
